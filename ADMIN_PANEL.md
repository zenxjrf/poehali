# 🛡️ Админ-панель Poehali

## Доступ к админ-панели

Админ-панель доступна **только** для пользователей с указанными Telegram ID.

### Как добавить администратора

1. Откройте `frontend/src/App.jsx`
2. Найдите строку:
   ```javascript
   const ADMIN_TELEGRAM_IDS = ['123456789', '987654321']
   ```
3. Добавьте ваш Telegram User ID в массив

### Как узнать свой Telegram ID

1. Откройте бота [@userinfobot](https://t.me/userinfobot)
2. Нажмите `/start`
3. Бот отправит ваш User ID (число)
4. Скопируйте ID и добавьте в `ADMIN_TELEGRAM_IDS`

---

## Как использовать

1. Откройте Web App через Telegram бота
2. Если ваш Telegram ID есть в `ADMIN_TELEGRAM_IDS`, в правом верхнем углу появится кнопка **⚙️**
3. Нажмите на кнопку ⚙️ для открытия админ-панели

---

## Функции админ-панели

### 📊 Статистика
- Количество отзывов
- Количество заказов

### ⭐ Последние отзывы
- Просмотр последних 5 отзывов
- Рейтинг (звёзды)
- Имя клиента
- Текст отзыва
- Дата создания

### 🚕 Последние заказы
- Просмотр последних 5 заказов
- Номер заказа
- Статус заказа
- Имя и телефон клиента
- Количество пассажиров
- Комментарий

## Пример настройки

```javascript
// frontend/src/App.jsx
const ADMIN_TELEGRAM_IDS = [
  '123456789',  // Ваш ID
  '987654321',  // ID другого администратора
  '555666777'   // ID менеджера
]
```

---

## Безопасность

- ⚠️ **Не коммитьте реальные ID в репозиторий!**
- Используйте переменные окружения для продакшена:
  ```javascript
  const ADMIN_TELEGRAM_IDS = (import.meta.env.VITE_ADMIN_IDS || '123456789').split(',')
  ```
- Добавьте `.env` в frontend:
  ```
  VITE_ADMIN_IDS=123456789,987654321
  ```
